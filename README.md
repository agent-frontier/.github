# agent-frontier/.github

This repository contains shared community health and configuration files for the **agent-frontier** GitHub organization. Files in this repo are used to standardize workflows, policies, and contributor experience across the organization’s other repositories.

## What’s included

Common file types in this repository may include:
- **Issue templates** (bug reports, feature requests, questions)
- **Pull request templates**
- **Health files** (e.g., contributing guides, code of conduct)
- **Security policy** and/or disclosure instructions
- **CODEOWNERS** and/or review/approval configuration
- **GitHub workflows** or automation for org-wide settings
- **Documentation** for maintainers and contributors

## Usage

You typically do not need to clone this repository for development. Instead:
- Update templates/policies here to apply consistent guidance across all repos in the org.
- Refer to individual repositories for product-specific documentation and code.

## Contributing

If you’re proposing changes to the shared templates or policies:
1. Open a pull request against this repository.
2. Include a clear description of what you’re changing and why.
3. If adding/updating templates, ensure they align with the org’s current practices.
